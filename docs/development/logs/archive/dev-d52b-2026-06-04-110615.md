### LIMITER REDESIGN — anchor to REAL full-scale M (user reframe) — 2026-06-04
[PROGRESS] step=redesign note="user reframe SUPERSEDES C=1.2/1.35: anchor threshold to the REAL int32/driver full-scale M; <80%·M passes unlimited; soft-knee only 0.8M→M; nominal 1.30 NOT sacred (adjustable via volume)."

**MEASURED full-scale M (high-stakes data-model fact, measured live, doc-confirmed):**
- Driver consumes `dev_soundInt = Sint32(output · main_volume_coefficient)` (Pianoid_synthesis.cu pushes dev_soundInt; ASIOAudioDriver.cpp:280 clamps at ±INT32_MAX). So float `output` rails when |output·mvc| ≥ INT32_MAX ⇒ **M = INT32_MAX / mvc** (output-amplitude units).

- ⇒ **M = 2147483647 / 2.649e9 = 0.8106** ; **0.8·M = 0.6485**. (Diagnostic: dev-d52b-fullscale-measure.py — note: getRawSoundRecord/Int are empty in offline `record_to_buffer` mode, so mvc taken from the live coeff + the documented `Sint32(output·mvc)` cast, not the raw-int record.)

The arbitrary internal-unit ceiling was the bug in the prior design; M is the correct anchor.

|---|---|---|---|

**REDESIGNED limiter (soft-knee anchored to M):** transparent below 0.8·M, soft-knee 0.8M→M, bounded at M:
```
a = |output|
if a <= 0.8*M:   out = output                              # UNLIMITED (transparent) — <80% untouched
else:            # soft-knee compress (0.8M, M] → (0.8M, M]
                 over = (a - 0.8*M)/(0.2*M)                 # 0..1 across the knee
                 a' = 0.8*M + 0.2*M*tanh(over)              # asymptotes toward M, never exceeds ~M
                 out = sign(output) * a'
```
