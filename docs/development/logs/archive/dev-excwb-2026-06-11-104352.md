ROOT CAUSE (measured + code-confirmed, two compounding factors):
1. **Per-pitch fan-out (granular path applied to a whole vector).** `usePreset.changeParametersOfStrings`'s debounced inner LOOPS over every changed pitch firing one `set_parameter/string/<pitch>` WS emit each (usePreset.js:856-877). For a per-pitch CELL edit that's 1; for a Workbench VECTOR edit (function-apply / drag / extend-shrink) it's N (≈83 here, up to 88 / num-modes for the full range).
