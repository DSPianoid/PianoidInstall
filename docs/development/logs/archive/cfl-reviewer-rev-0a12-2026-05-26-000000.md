### Coefficient definitions (from doc, to re-derive)
- `coeff_tension` = `(T/ρ)·dt²/dx²`
- `coeff_bending` = `(π·E·r⁴/4ρ)·dt²/dx⁴`  (B)
- `dt = 1/(sample_rate·string_iteration)`
- `dx` set by string geometry (VERIFY how).
- shift_0 = (2 + 12B − 2T)·dec_inv
- shift_1 = (T − 8B)·dec_inv
- shift_2 = 2B·dec_inv
- shift_b = (dec_curr − 1)·dec_inv
- dec_inv = 1/(1 + dec_curr)
- coeff_frequency_decay present as separate HF term.

### Characteristic quadratic (doc):
g² − A(θ)g − B0(θ) = 0
A(θ) = shift_0 + 2·shift_1·cosθ + 2·shift_2·cos2θ + coeff_frequency_decay·(−2)(1−cosθ)
B0(θ) = shift_b + coeff_frequency_decay·2(1−cosθ)

## [PROGRESS] Phase 1 — Math review DONE (independent re-derivation)

Ran my own vectorized derivation (`dev-cfl-rev0a12-fast.py`) + reran dev-cflfix's
(`dev-cfl-vonneumann-derivation.py`, `dev-cfl-complete-condition.py`). Findings:

**Q1 — closed-form bound CONFIRMED.** Excluding DC, exact stable band = `8B ≤ T ≤ 1+8B` to 6
digits for B∈[0,0.05] (drifts only at B≥0.1, far outside presets). Velocity damping `dec_curr`
does NOT move edges (T_hi−8B=1.000 ∀ dec∈[0,2]). HF damping `cfd` TIGHTENS upper edge
(cfd=0.1 → T_hi−8B=0.80; cfd=0.5 → 0.0). Matches SYNTHESIS_ENGINE.md exactly.

**Q2 — binding θ: doc is IMPRECISE.** As T→1⁻ (approaching upper CFL edge from BELOW), binding
θ → **0⁺** (long wavelength near DC), NOT Nyquist. Measured T=0.99,B=0 → max|g| at θ=0.0008 rad.
Nyquist θ=π binds the LOWER edge (T<8B) and the overshoot once already unstable (T=1.01→π).
So "binding wavenumber = Nyquist θ=π" conflates two modes. THIS is why dev-cflfix saw "interior-θ
instabilities" and built a 48-pt sweep. BUT see Q4 — the Jury closed form removes the need to sweep.

**Q3 — θ=0 exclusion JUSTIFIED (resolved, symbolic proof).** A(0)=(2+12B−2T)+2(T−8B)+2(2B)=2
EXACTLY ∀T,B (T,B terms cancel). B0(0)=shift_b=−1 (undamped). ⇒ g²−2g+1=0 ⇒ g=1 DEFECTIVE double
root (rigid-body translation; fixed-fixed boundary pins it — not a free mode). Live |g(0)|=1.00042
is FLOAT ROUNDOFF on this exact double root (disc=A²+4B0=0 exactly, flips +ε numerically). Excluding
θ=0 is NOT a damping-discretization hack at DC. dev-cflfix was RIGHT to exclude it. (Subtlety: the
right reason is "exactly-known defective double root", not "carries no wave info".)

**Q4 — THE SIMPLICITY RESULT.** Closed-form Jury/Schur test on final coeffs reproduces exact max|g|
with **0 mismatches** incl. damping (undamped, dec=0.3, cfd=0.1; 1519-cell grids). AND the
**Nyquist-only (θ=π) Jury check = 0/651 mismatches across the ENTIRE physical regime** (T≤0.05,
B≤0.001). Only mismatches in the WIDE grid at large NEGATIVE B (physically impossible).
⇒ A simple closed-form gate is viable and EXACT. The 48-pt θ-sweep + companion matrix is unnecessary
machinery for the physical regime. (dev-cflfix's own dev-cfl-complete-condition.py PART2/PART3 found
the same: Jury 0/1763, endpoints-only 0/1763.)

**Q(d) eps=1e-6 — JUSTIFIED as float-precision guard, NOT a fudge.** The criterion is exact (|g|≤1);
eps only absorbs ~1e-7 round-off. The 1e-3 push was a (correctly-reverted) band-aid. With the
Jury closed form + DC excluded, the spurious-overshoot source that motivated widening is GONE
(the DC defective root was the overshoot source), so even eps can shrink toward pure round-off.
