### Backend (PianoidCore)

- **NEW:** `pianoid_middleware/modal_adapter/external_export.py` (~430 LOC)
  - `extract_signed_shapes_per_chain` — per-chain projection of complex
    per-channel mode shapes onto the chain's reference shape, with response-
    channel restriction (excludes force/reference channels), per-chain
    max-abs normalisation. Also returns per-chain amplitude payloads
    (loudest detection's complex shape per chain) for `decka_coeff`.
  - `approximate_line` — linear `scipy.interpolate.interp1d` matching
    Stage-2's behaviour (extrapolate outside measured range, broadcast
    when only 1 point is available).
  - `approximate_planar` — least-squares plane fit `z = a*x + b*y + c`
    via `np.linalg.lstsq` on populated cells. Falls back to mean for
    fewer than 3 cells. Measured cells are preserved exactly (only holes
    are filled with the planar evaluation).
  - `build_export_payload` — orchestrates: chain filter by selection,
    sort by frequency, signed-shape projection, layout-aware approximation,
    pad to `MAX_MODES=128` placeholders, build `Ci_coef_cos` / `omega` /
    `Q` / `decka_coeff` arrays + sidecar.
  - `write_export_files` — `np.savetxt` with `fmt="%.6f", delimiter="\t"`
    matching Stage-2 byte-for-byte.
  - `export_text_files` — top-level convenience.
- **MODIFIED:** `pianoid_middleware/modal_adapter/modal_adapter.py` (+72 LOC)
  - New method `ModalAdapter.export_to_text_files(output_dir=None,
    selected_chain_ids=None)` appended at end of class. Default output =
    `{project_dir}/modal_adapter/export_text/`. Validates project + chains
    + mapping pre-conditions.
- **MODIFIED:** `pianoid_middleware/modal_adapter/routes.py` (+57 LOC)
  - New route `POST /modal/projects/<name>/export_text`. Validates that
