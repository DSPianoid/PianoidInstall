The Problem

Why do some pianos sing while others stay dull — and why can't we answer this systematically?

Systematic soundboard design is economically and methodologically impossible with current tools. A concert grand has 230 strings coupled through a soundboard with thousands of resonance modes. The interactions between these modes determine the instrument's character — but evaluating a single design change requires weeks of skilled labor and thousands in materials.

  • "Killer octave" — poor sustain in 5th–6th octave due to modal density and impedance mismatch. The mechanism is understood; the design solution is not
  • Design knowledge exists as intuition accumulated over decades, not as transferable, quantifiable parameters

Research question:
Can a closed-loop methodology — FEA design, real-time acoustic preview, physical fabrication, and quantified modal verification — make soundboard design systematic and reproducible?

The core experiment: A/B synthesis — FEA-predicted modes vs. physically measured modes on the same engine, isolating model accuracy from synthesizer fidelity. If the predicted and measured soundboards converge perceptually, the design loop is validated.

What Already Exists: The Pianoid Platform

Working System — demonstrated at NAMM Show and in live concert performances

  • Synthesis algorithm — full string set modelled with waveguide model, 256 soundboard modes coupled to all strings simultaneously, running real-time on NVIDIA 4070 Ti with 30–50% safety margin
  • Full inter-string coupling via feedin/feedback matrices (how each soundboard mode is energised by string motion and returns energy back) — sympathetic resonance, pedal effects emerge naturally
  • Patented excitation system — 8-channel electromechanical actuators, tested and proven on a real piano soundboard
  • Modal measurement pipeline — multi-band ESPRIT extraction with automated mode tracking across 78+ bridge positions

What This Proposal Builds

  • FEA import pipeline — extract feedin/feedback matrices from FEA eigenmode output
  • Acoustic radiation model — model the sound wave at the listening point outside the soundboard
  • Algorithm improvement — 256 → 2048 modes (already in pipeline), increased string discretisation, enhanced hammer excitation model
  • Soundboard fabrication — rapid prototyping pipeline, testing of materials, structural design of the encasing
  • Perceptual validation protocol — ABX listening tests, comparative evaluation

How We Enable the Loop

Closing the design loop requires two capabilities that conventional piano tools cannot provide:

Real-Time Polyphonic Preview

GPU modal synthesis with full inter-string coupling lets a pianist play the designed soundboard — not listen to single offline notes. Design changes are heard instantly. Iteration takes minutes, not weeks.

Patented Excitation System

Electromechanical actuators driven by GPU-computed string physics replace physical strings entirely. No permanent tension, no iron plate, no frame. Soundboards are swappable — free mounting with no stress. A new design can be tested acoustically in hours.

Together, these make the FEA → preview → fabricate → measure → verify cycle practically executable.

Loop closure criterion: quantified modal correlation (frequency, MAC, damping) defines go/no-go. A/B perceptual test validates convergence.

Phases 1–2: Design & Preview

FEA eigenvalue analysis (COMSOL/Abaqus). Per mode: frequency, quality factor, modal mass, mode shape amplitudes at string/actuator positions. Mode shapes define the feedin/feedback coupling matrices. Import into Pianoid for real-time polyphonic preview.

Phase 3: Fabrication

Reduced-scale test boards for rapid iteration. Material options: spruce, plywood, composites. Carbon fiber ribs are a hypothesis to test, not a proven approach.

Phase 4: Modal Verification

Multi-channel ESPRIT + automated mode tracking. FEA-vs-measured: frequency, MAC (Modal Assurance Criterion), damping.

Phase 5: A/B Synthesis & Evaluation

The core experiment: same engine, FEA modes vs. measured modes. Isolates model accuracy from synthesizer fidelity. ABX listening tests, expert musician assessment.

Novel Design Opportunities

Removing string tension and fixed bridge geometry opens a design space that is inaccessible within the conventional piano paradigm:

Soundboard Freedom

  • Arbitrary geometry — shape, curvature, and thickness profile optimised for target modal properties, not constrained by a piano case
  • Any material — spruce, composites, engineered wood, sandwich panels. Acoustic viability tested computationally before fabrication
  • Free mounting — no permanent stress or tension. Boundary conditions become a design variable

Excitation Freedom

  • Multi-point excitation — actuators at any positions across the surface, not confined to a linear bridge
  • Frequency-optimised placement — excitation points chosen to maximise coupling to target modes
  • Programmable coupling topology — feedin/feedback matrices can implement coupling patterns impossible with physical strings

These opportunities are consequences of the closed-loop platform — explored systematically using the same FEA → preview → verify methodology.

Why Now?

Technology Convergence

  • GPU compute is crossing the threshold — real-time synthesis featuring waveguide strings and full modal feedback loop is feasible for the first time
  • MAESSTRO laid the FEA groundwork — soundboard modal synthesis is proven feasible; the missing piece is real-time performance and a verification loop
  • ESPRIT / modal analysis tools are mature — high-resolution spectral methods now extract hundreds of modes reliably

Platform Readiness

  • Pianoid is demonstrated, not hypothetical — engine, patented excitation system, and measurement pipeline shown at NAMM and in live concert performances
  • The closed-loop concept is now executable — the required combination of technologies was not available five years ago

Growing Demand

  • Sustainable materials research — creating new candidate soundboard materials that need acoustic evaluation tools
  • Digital twin methodology — established in aerospace and automotive; applying it to musical instruments is timely

Key Risks

2048-mode scaling fails

Coupling flattening invalid or register pressure too high. Fallback: 256-mode model (proven timbre). Next-gen NVIDIA adds headroom.

Poor FEA-to-measured correlation

First prototype modes don't match. Mitigation: start with simple geometry. Discrepancies inform model refinement — this is the research.

Prototype acoustic validity

Non-traditional materials may not be valid proxies. Mitigation: spruce baseline alongside experimental materials.

Partnership timeline

Academic schedules, facility access, PI availability. Mitigation: parallel workstreams; industry-side work proceeds independently months 1–3.
