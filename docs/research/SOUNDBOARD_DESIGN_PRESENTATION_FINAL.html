The Problem

Why do some pianos sing while others stay dull — and why can't we answer this systematically?

Systematic soundboard design is economically and methodologically impossible with current tools. A concert grand has 230 strings coupled through a soundboard with thousands of resonance modes. Evaluating a single design change requires weeks of skilled labor and thousands in materials.

  • "Killer octave" — poor sustain in 5th–6th octave due to modal density and impedance mismatch. The mechanism is understood; the design solution is not
  • Design knowledge exists as intuition accumulated over decades, not as transferable, quantifiable parameters

Research question:
Can a closed-loop methodology — FEA design, real-time acoustic preview, physical fabrication, and quantified modal verification — make soundboard design systematic and reproducible?

The core experiment: A/B synthesis — FEA-predicted modes vs. physically measured modes on the same engine, isolating model accuracy from synthesizer fidelity. If the predicted and measured soundboards converge perceptually, the design loop is validated.

What Already Exists: The Pianoid Platform

Working System — demonstrated at NAMM Show and in live concert performances

  • Synthesis algorithm — full string set modelled with finite-difference wave equation solver, 256 soundboard modes coupled to all strings simultaneously, real-time on NVIDIA 4070 Ti with 30–50% safety margin
  • String-mode coupling network — each soundboard mode is energised by string motion and returns energy back, creating continuous bidirectional exchange between all strings and all modes. Sympathetic resonance, pedal effects emerge naturally
  • Patented excitation system — 8-channel electromechanical actuators, tested and proven on a real piano soundboard
  • Modal measurement pipeline — multi-band ESPRIT extraction with automated mode tracking across 78+ bridge positions

What This Proposal Builds

  • FEA import pipeline — extract string-mode coupling matrices from FEA eigenmode output
  • Mode capacity: 256 → 2048 — already in pipeline. Flatten coupling coefficients for higher modes
  • String discretisation improvement — higher-resolution finite-difference string model
  • Enhanced hammer excitation model — more accurate initial transient
  • Soundboard fabrication pipeline — rapid prototyping, material testing, encasing design
  • Perceptual validation protocol — ABX (forced-choice perceptual comparison) listening tests, comparative evaluation

How We Enable the Loop

Closing the design loop requires two capabilities that no existing tool provides:

Real-Time Polyphonic Preview

GPU modal synthesis with full string-mode coupling lets a pianist play the designed soundboard — not listen to single offline notes. Design changes are heard instantly. Iteration takes minutes, not weeks.

Patented Excitation System

Electromechanical actuators replace physical strings entirely. No permanent tension, no iron plate, no frame. Soundboards are swappable — free mounting with no stress. A new design can be tested acoustically in hours.

Together

These make the FEA → preview → fabricate → measure → verify cycle practically executable.

Beyond the loop

Freeing the soundboard from its structural role also opens the door to designs that no conventional piano could accommodate.

Loop closure: quantified modal correlation (frequency, MAC, damping) defines go/no-go. A/B perceptual test validates convergence.

Phases 1–2: Design & Preview

FEA eigenvalue analysis (COMSOL/Abaqus). Per mode: frequency, quality factor, modal mass, mode shape amplitudes at excitation positions. Mode shapes define the string-mode coupling matrices. Import into Pianoid for real-time polyphonic preview.

Phase 3: Fabrication

Reduced-scale test boards for rapid iteration. Material options: spruce, plywood, composites. Carbon fiber ribs are a hypothesis to test, not a proven approach.

Phase 4: Modal Verification

Multi-channel ESPRIT + automated mode tracking. FEA-vs-measured: frequency, MAC (Modal Assurance Criterion), damping.

Phase 5: A/B Synthesis & Evaluation

The core experiment: same engine, FEA modes vs. measured modes. Isolates model accuracy from synthesizer fidelity. ABX listening tests, expert musician assessment.

Why Now?

Technology Convergence

  • GPU compute is crossing the threshold — real-time synthesis with finite-difference string modelling and full string-mode coupling is feasible for the first time
  • MAESSTRO laid the FEA groundwork — soundboard modal synthesis is proven; the missing piece is real-time performance and a verification loop
  • Modal analysis tools are mature — ESPRIT and related methods now extract hundreds of modes reliably

Platform Readiness

  • Pianoid is demonstrated — patented excitation system, engine, and measurement pipeline shown at NAMM and in live concert performances
  • The unconstrained design concept is now executable — the required combination of real-time synthesis, FEA import, modal measurement, and excitation hardware has only recently become available

Growing Demand

  • Sustainable materials — new candidate soundboard materials need acoustic evaluation tools
  • Digital twin methodology — established in aerospace and automotive; extending it to acoustic instruments is timely

Key Risks

Algorithm improvements fail

Mode scaling, string discretisation, or hammer model enhancements exceed real-time budget or prove unstable. Fallback: current algorithm stays as-is — already demonstrated in live performances and proven adequate for the research.

Poor FEA-to-measured correlation

First prototype modes don't match. Mitigation: start simple. Discrepancies inform refinement — this is the research.

Prototype acoustic validity

Non-traditional materials as proxies. Mitigation: spruce baseline alongside experimental materials.

Partnership timeline

Academic schedules, facility access. Mitigation: parallel workstreams; industry work proceeds independently months 1–3.
